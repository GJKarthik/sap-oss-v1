\begin{abstract}
\noindent
This document provides the detailed specification for AI-augmented Trial Balance (TB) review,
covering the month-end close process across 234 legal entities. It consolidates the business
requirements, formal process specification (state machine with 24 states and 5 decision gateways),
training data design from the HKG TB/PL workbooks, and data schema definitions. The specification
targets 5 FTE savings through automated extraction, ML-based variance analysis, LLM-generated
commentaries, and FORTM pack narrative generation, with a foundation for daily control operations.
\end{abstract}

\section{Business Requirements}
\label{sec:requirements}

\subsection{Executive Summary}

Trial Balance review is conducted during M+1 to M+7 across all Legal entities, Segment \& Group,
to understand variance \& balance movements between periods. An AI model produces business insights
and written commentaries based on GL transaction data analysis---detecting anomalies, trends, unusual
movements, new accounts, spikes/drops vs prior periods, and material balances.

\section{Process Specification}
\label{sec:process}

\subsection{Overview}

The TB review follows a formal BPMN workflow (source: ``Operation of Month End Close Controls -- Trial Balance Review -- GLO'', Process Owner: Modi, Piyush). The workflow comprises \textbf{24 states}, \textbf{5 exclusive gateways}, and spans \textbf{4 timeline phases}.

\subsection{Timeline Phases}

\begin{table}[H]
\centering
\caption{Process Timeline}
\begin{tabularx}{\textwidth}{llX}
\toprule
\textbf{Phase} & \textbf{Timing} & \textbf{Description} \\
\midrule
Preparation  & M-3           & Setup: roll forward file, update parameters and master data \\
Extraction   & M-3 to M+5    & Extract TB from PSGL/S4 and load into variance analysis file \\
Analysis     & M-2 to M+5    & Variance calculation, commentary collection, investigation, risk \\
Review       & M+5           & Internal review, incorporate changes, send FORTM pack \\
\bottomrule
\end{tabularx}
\end{table}

\subsection{State Machine}

The complete workflow is defined as a state machine. Each state has a type (task, gateway, event), timing, associated systems, and transitions.

\subsubsection{Preparation Phase (M-3)}

\begin{enumerate}[nosep]
    \item \textbf{START} $\rightarrow$ Prepare Schedule Work Day
    \item \textbf{ROLL\_FORWARD} -- Roll Forward Prior Month File {\small [MS Excel, Shared Drive]}
    \item \textbf{DOWNLOAD\_PSGL} -- Download PSGL Master Account File {\small [PSGL, MS Excel]}
    \item \textbf{UPDATE\_ACCOUNTS\_MASTER} -- Update Accounts Master Sheet {\small [MS Excel]}
    \item \textbf{UPDATE\_PARAMETERS} -- Update Parameters \& Comparatives {\small [MS Excel]}
\end{enumerate}

\subsubsection{Extraction Phase (M-3 to M+5)}

\begin{enumerate}[nosep,start=6]
    \item \textbf{EXTRACT\_TB} -- Extract Trial Balance {\small [SC Bridge, MS Excel]}
    \item \textbf{UPDATE\_TB\_DETAILS} -- Update TB Details in Variance File {\small [MS Excel, Shared Drive]} \\
          {\footnotesize EUC: EX/200/1579702/0001/000}
\end{enumerate}

\subsubsection{Analysis Phase (M-2 to M+5)}

\begin{enumerate}[nosep,start=8]
    \item \textbf{CALCULATE\_VARIANCE} -- Calculate Period-over-Period Variance {\small [MS Excel]}
    \item \textbf{FILTER\_VARIANCES} -- Filter by Materiality Criteria {\small [MS Excel]}
    \item \textbf{CHECK\_MATERIAL\_VARIANCES} {\small [Gateway]} -- Material?
        \begin{itemize}[nosep]
            \item Yes $\rightarrow$ SEEK\_COMMENTARIES
            \item No $\rightarrow$ CHECK\_UNEXPLAINED
        \end{itemize}
    \item \textbf{SEEK\_COMMENTARIES} -- Request Explanations {\small [MS Outlook]}
    \item \textbf{RECEIVE\_COMMENTARIES} -- Collect Responses {\small [Event]}
    \item \textbf{UPDATE\_COMMENTARIES} -- Record in Variance File {\small [MS Excel]}
    \item \textbf{IS\_JOURNAL\_POSTED} {\small [Gateway]} -- Journal Status?
        \begin{itemize}[nosep]
            \item Posted $\rightarrow$ CHECK\_UNEXPLAINED
            \item Required $\rightarrow$ SEND\_JOURNAL\_REQUEST
            \item Not Posted $\rightarrow$ CHECK\_FURTHER\_INVESTIGATION
        \end{itemize}
    \item \textbf{SEND\_JOURNAL\_REQUEST} -- Request Posting {\small [MS Outlook]}
    \item \textbf{CHECK\_FURTHER\_INVESTIGATION} {\small [Gateway]} -- Investigate?
        \begin{itemize}[nosep]
            \item No $\rightarrow$ CHECK\_UNEXPLAINED
            \item Yes $\rightarrow$ CHECK\_ENTRIES\_STATUS
        \end{itemize}
    \item \textbf{CHECK\_ENTRIES\_STATUS} -- Follow Up with Teams {\small [MS Outlook]}
    \item \textbf{CHECK\_CONFIRMATION} {\small [Gateway]} -- Confirmed by M+4?
        \begin{itemize}[nosep]
            \item Yes $\rightarrow$ CHECK\_UNEXPLAINED
            \item No $\rightarrow$ TRACK\_ENTRIES
        \end{itemize}
    \item \textbf{TRACK\_ENTRIES} -- Track and Risk Analysis {\small [MS Excel]}
    \item \textbf{UPDATE\_RISK} -- Update Risk Assessment {\small [MS Excel]}
    \item \textbf{CHECK\_UNEXPLAINED} {\small [Gateway]} -- Unexplained Variances?
        \begin{itemize}[nosep]
            \item None $\rightarrow$ PREPARE\_SUMMARY
            \item Yes $\rightarrow$ CATEGORIZE\_UNEXPLAINED
        \end{itemize}
    \item \textbf{CATEGORIZE\_UNEXPLAINED} -- Classify Unexplained Variances {\small [MS Excel]}
\end{enumerate}

\subsubsection{Review Phase (M+5)}

\begin{enumerate}[nosep,start=23]
    \item \textbf{PREPARE\_SUMMARY} -- Consolidate Analysis {\small [MS Excel]}
    \item \textbf{SEND\_INTERNAL\_REVIEW} -- Submit for Review {\small [MS Outlook, Shared Drive]}
    \item \textbf{REVIEW\_INCORPORATE} -- Incorporate Changes {\small [MS Excel, Shared Drive]} \\
          {\footnotesize EUC: EX/200/1579702/0002/000}
    \item \textbf{SEND\_FORTM} -- Submit FORTM Pack {\small [MS Outlook, Shared Drive]}
    \item \textbf{END} -- TB Review Completed for Month
\end{enumerate}
