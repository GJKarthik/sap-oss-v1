\subsection{TB Review Spec Integration}

This HITL traceability specification is a companion document to the
main TB Review Specification (\texttt{tb-review-spec.tex}). Key
integration points:

\begin{itemize}[noitemsep]
  \item Chapter references use the \texttt{tb:} prefix for cross-document links
  \item Specification IDs (\specref{TB-STEP-*}, \specref{DP-*}, etc.) are
        defined in the TB Review Spec
  \item Schema definitions are shared via the common schema registry
\end{itemize}
