\section{Auditor Guidance}

\subsection{Evaluating Direct Implementations}

For direct implementations, auditors should verify:
\begin{itemize}[nosep]
    \item Constants match exactly (e.g., 1.4826, 3.0, 3.5)
    \item Formulas are mathematically equivalent
    \item Threshold comparisons use same direction ($>$, $<$, $\geq$, $\leq$)
\end{itemize}

\subsection{Evaluating Conceptual Adaptations}

For conceptual adaptations, auditors should verify:
\begin{itemize}[nosep]
    \item The underlying principle is preserved
    \item The adaptation rationale is documented
    \item The implementation achieves the same goal in the new domain
\end{itemize}

\subsection{Evaluating Inherited Implementations}

For inherited implementations, auditors should verify:
\begin{itemize}[nosep]
    \item The underlying framework implements the concept correctly
    \item The SAP-OSS configuration enables the feature
    \item Version compatibility is maintained
\end{itemize}
