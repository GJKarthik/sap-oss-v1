\chapter{Code Reference Guide}
\label{chap:code-reference-guide}

This appendix provides a guide to the SAP-OSS codebase locations that implement IP concepts, including directory structure, key files, and navigation guidance for auditors.

\section{Codebase Overview}

The SAP-OSS codebase is organized under the \filepath{src/} directory:

\begin{verbatim}
src/
├── intelligence/           # AI core services
│   ├── ai-core-pal/       # Governance, agents, PAL service
│   │   ├── governance/    # IP-001, IP-002 implementations
│   │   └── data_products/ # ODPS compliance registry
│   ├── analytics/         # IP-004, IP-007 implementations
│   ├── vllm-turboquant/   # IP-003 inference safety
│   └── ocr/               # Document processing
├── training/              # Training pipelines (IP-005)
├── generativeUI/          # UI components (IP-006)
└── data/                  # LangChain, OData integration
\end{verbatim}

\section{Code Navigation for Auditors}

\subsection{Finding IP Concept Implementations}

To locate implementations of specific IP concepts:

\begin{enumerate}[nosep]
    \item \textbf{Search for IP references in comments}: Many implementations include IP document references in docstrings:
    \begin{verbatim}
    grep -r "IP-004" src/
    grep -r "Section 7" src/intelligence/analytics/
    \end{verbatim}
    
    \item \textbf{Search for specific constants}:
    \begin{verbatim}
    grep -r "1.4826" src/   # MAD constant
    grep -r "ZSCORE_THRESHOLD" src/
    \end{verbatim}
    
    \item \textbf{Search for function names}:
    \begin{verbatim}
    grep -r "_mad_detection" src/
    grep -r "detect_spike_drop" src/
    \end{verbatim}
\end{enumerate}

\subsection{Verification Checklist}

For each IP concept implementation, auditors should verify:

\begin{itemize}[nosep]
    \item[$\square$] Code file exists at documented location
    \item[$\square$] Function/class name matches documentation
    \item[$\square$] Constants match IP specification values
    \item[$\square$] Algorithm logic matches IP description
    \item[$\square$] Docstrings reference IP source (where applicable)
\end{itemize}
